# lamp-blueprint

This is a simple blueprint setting up a LAMP instance - Linux + Apache + MariaDB + PHP.

By default the instance has following software:
  - Debian Stretch
  - Apache 2.4
  - MariaDB 10.1
  - PHP-fpm 7.0
 
 Defaut Apache MPM:
  - mpm_event
 
 PHP Modules:
  - mbstring
  - mcrypt
  - mysql
  - zip
  - xml
  - gd
  - curl
