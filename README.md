# Subutai Moodle Blueprint

This is a simple blueprint setting up a Moodle site on top of a LAMP instance - Linux + Apache + MariaDB + PHP.

By default the instance has following software:
  - Debian Stretch
  - Apache 2.4
  - MariaDB 10.1
  - PHP-fpm 7.0
  - Moodle
 
 Defaut Apache MPM:
  - mpm_event
 
 PHP Modules:
  - mbstring
  - mcrypt
  - mysql
  - zip
  - xml
  - xmlrpc
  - intl
  - opcache
  - gd
  - curl
  - json
